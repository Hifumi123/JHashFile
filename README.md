# JHashFile

JHashFile 是使用 Java 开发的文件校验工具。

## 开发环境

* Java 17
* Eclipse 2023-03 R
* Maven 3.6.3

## 支持的算法

* MD5
* SHA1
* CRC32

## 依赖

* [Apache Commons Codec](https://commons.apache.org/proper/commons-codec)
* [JavaFX](https://openjfx.io)
* [JUnit 5](https://junit.org/junit5)

详见 POM 文件。
